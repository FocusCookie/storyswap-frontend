# Button

## Displays an image that represents a user or organization

<bg-red-500>Use an avatar for attributing actions or content to specific users. HIER</bg-red-500>

The user's name should _always_ be present when using Button – either printed beside the avatar or in a tooltip.

export const Template = (args) => <Button {...args} />;

<Story
name="standard"
args={{
    size: "large",
    username: "Tom Coleman",
    src: "https://avatars2.githubusercontent.com/u/132554",
  }}

> {Template.bind({})}
> </Story>

### Sizes

4 sizes are supported.

  <div>

  </div>
</Story>

### Default Values

When no image is supplied to the `src` prop, Button displays initials.

Button should be used sparingly in situations without access to images.

<Story name="initials">
  <div>
    <Button username="Tom Coleman" />
    <Button username="Dominic Nguyen" />
    <Button username="Kyle Suss" />
    <Button username="Michael Shilman" />
  </div>
</Story>

### Loading

The loading state is used when the image or username is, well, loading.

<Story name="loading">
  <div>
    <Button size="large" loading />
    <Button size="medium" loading />
    <Button size="small" loading />
    <Button size="tiny" loading />
  </div>
</Story>

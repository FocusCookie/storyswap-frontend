# Colors

This project uses Tailwindcss so a lot of colour props are passed in as CSS classes that
Tailwind applies styling to i.e. `text-blue-500` to use the primary blue. You can find the
full colour palette [here](https://tailwindcss.com/docs/customizing-colors/#default-color-palette). The custom colors below can be applied in the same way as the standard tailwind colors i.e. `bg-primary-500`.
